## Correr el proyecto localmente

### Clonar el proyecto

```bash
  git clone https://github.com/LukaLiuzzi/rockanddolls-react
```

Ir al repositorio

```bash
  cd rockanddolls-react
```

Instalar dependencias

```bash
  npm install
```

Iniciar el servidor de desarrollo

```bash
  npm run start
```
